# Nexora - ADHD Support Platform

A modern, professional platform designed to help people with ADHD manage tasks, build sustainable habits, and unlock their full potential.

## Quick Start

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

### Backend Setup
```bash
cd backend
npm install
npm run dev
```

Server runs on [http://localhost:5000](http://localhost:5000)

## Features

- Smart task management with AI assistance
- Focus sessions and habit tracking
- AI-powered insights
- Distraction blocker
- Community support
- Team collaboration

## Design

- Color Palette: Deep Navy, Cream, Sage Green
- Sharp edges, no rounded corners
- Minimal animations
- Editorial, sleek style
